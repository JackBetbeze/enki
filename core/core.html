core — Instances of core classes, initialize and terminate the system¶

Module initializes system at startup, terminates it, when mksv3 closed, and used for get core instances, such as main window, workspace, etc.

class mks.core.core.Core¶

Core object initializes system at startup and terminates at close.

It creates instances of other core modules and holds references to it

init()¶

Initialize core.

Called only by main()

Called only by main()
