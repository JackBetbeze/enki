core — Instances of core classes, initialize and terminate the system¶

Module initializes system at startup, terminates it, when mksv3 closed, and used for get core instances, such as main window, workspace, etc.

class mks.core.core.Core¶

Core object initializes system at startup and terminates when closing.

It creates instances of other core modules and holds references to it

init()¶

Initialize core.

Called only by main()

term()¶

Terminate plugins and core modules

Called only by main()

mainWindow()¶

Get mks.core.mainwindow.MainWindow instance

actionModel()¶

Get main window action model instance

workspace()¶

Get mks.core.workspace.Workspace instance

config()¶

Get mks.core.config.Config instance

messageToolBar()¶

Get queued message bar instance

loadedPlugins()¶

Get list of curretly loaded plugins (:mks.core.Plugin instances)

setIndentHelper(language, indentHelper)¶

Set :mks.core.abstractdocument.IndentHelper for language. Pass None to clear the value

indentHelper(language)¶

Get :mks.core.abstractdocument.IndentHelper for the language

Raises KeyError, if not available

mks.core.core.core¶

Core instance. It is accessible as:

from mks.core.core import core core.anyMethod()
