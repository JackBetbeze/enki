monkeycore — Get instances of core classes, initialize and terminate the system¶

Module initializes system at startup, terminates it, when mksv3 closed, and used for get core instances, such as main window, workspace, etc.

mks.monkeycore.init()¶

Initialize the system, create main window, load plugins.

Called by main()

mks.monkeycore.term()¶

Terminate plugins and core modules

Called by main()

mks.monkeycore.mainWindow()¶

Get mks.mainwindow.MainWindow instance

Instance created, if not exists yet

mks.monkeycore.menuBar()¶

Get main window menu bar.

Instance created, if not exists yet

mks.monkeycore.workspace()¶

Get mks.workspace.Workspace instance

Instance created, if not exists yet
