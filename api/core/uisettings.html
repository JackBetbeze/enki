uisettings — Settings dialogue¶

Module provides GUI to edit settings. This GUI may be used by other core modules and by plugins.

Conception¶

Settings dialogue subsystem consists of 3 major entities:

  • UISettings.ui GUI form. Contains of controls.

  • *Option classes.

    Every object of the class links together control on GUI and option in the config file. It loads its option from config to GUI, and saves from GUI to config.

    config may be either mks.core.config.Config or python dictionary

  • mks.core.uisettings.ModuleConfigurator interface. Must be implemented by plugin or core module. Creates and holds *Option objects, applies module settings, when necessary. Flushes config.

GUI dialog invocation workflow¶

  1. MkS starts. Every plugin registers its ModuleConfigurator
  2. An user clicks Settings->Settings
  3. UISettings.ui are created
  4. mks.core.uisettings.UISettingsManager calls every ModuleConfigurator to load options
  5. ModuleConfigurator creates options. Every option loads its value from the mks.core.config.Config
  6. The user edits settigns
  7. The user clicks OK
  8. mks.core.uisettings.UISettingsManager calls every ModuleConfigurator to save settings
  9. ModuleConfigurator calls every option to save settings
  10. mks.core.uisettings.UISettingsManager calls every ModuleConfigurator to apply settings
  11. ModuleConfigurator applies module specific settings

Adding new settings¶

If you need to add own settings to UISettings dialog, you should:

  1. Implement and register your ModuleConfigurator
  2. Add controls to the dialog. You may edit UISettings.ui or add your controls dynamically during dialog creation (in ModuleConfigurator.__init__())
  3. Create *Option class instance for every configurable option.

Classes¶

Main classes:
  • mks.core.uisettings.UISettings - settings dialogue
  • mks.core.uisettings.ModuleConfigurator - plugin configurator
Classes for options:
  • mks.core.uisettings.CheckableOption - bool option, CheckBox
  • mks.core.uisettings.TextOption - string option, line edit
  • mks.core.uisettings.ListOnePerLineOption - list of strings option, text edit
  • mks.core.uisettings.NumericOption - numeric option, any numeric control
  • mks.core.uisettings.ColorOption - color option, button
  • mks.core.uisettings.FontOption - font option, button
  • mks.core.uisettings.ChoiseOption - string from the set option, combo box
