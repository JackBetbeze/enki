mks.plugins.schemeindenthelper.nextLineIndent(text)¶

Parse Scheme source code and suggest indentation for the next line Function raises UserWarning, if failed to parse the source

>>> nextLineIndent( '(myfunc a' )
8
>>> nextLineIndent( '(myfunc (if (a) a b)' )
8
>>> nextLineIndent( '(myfunc a)' )
0
>>> nextLineIndent( '  (myfunc a)' )
2
>>> nextLineIndent( '    (define' )
5
>>> nextLineIndent( 'a\nb' )
0
>>> nextLineIndent( '  a\n\n' )
2
>>> nextLineIndent( '(define myfunc' )
2
>>> nextLineIndent( '(let ((pi 3.14) (r 120))' )
2
