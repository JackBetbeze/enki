mainwindow — Main window of the UI. Fills main menu.¶

Module contains mks.mainwindow.MainWindow implementation

class mks.mainwindow.MainWindow¶

Bases: PyQt4.fresh.pMainWindow

Main UI window

Class creates window elements, fills main menu with items.

If you need to connect to some existing menu item triggered() signal - check action name in the class constructor, than use:

mks.monkeycore.menuBar().action( "mFile/aOpen" ).triggered.connect(self.myCoolMethod)

MainWindow instance accessible as

mks.monkeycore.mainwindow()
